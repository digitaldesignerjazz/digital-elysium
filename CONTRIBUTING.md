# Contributing to Digital Elysium

Thank you for your interest in building the Digital Elysium with us!

The Elysium Foundation is an ambitious initiative to create resilient decentralized infrastructure, ethical AI systems, transparent impact mechanisms, and regenerative technologies that help humanity reach its highest potential.

## How to Contribute

### Reporting Issues
- Use clear, descriptive titles
- Include steps to reproduce, expected vs actual behavior
- Suggest solutions or related work if possible

### Pull Requests
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-idea`)
3. Commit your changes with clear messages
4. Push to your fork
5. Open a Pull Request against `main`

We especially welcome contributions in these areas right now:
- Mesh networking tooling and deployment scripts (Yggdrasil, custom protocols)
- AI agent frameworks and swarm orchestration
- Blockchain primitives for impact tracking and community currencies
- Documentation, whitepapers, and strategic writing
- Prototyping sustainable / regenerative systems

### Code Style & Principles
- Prioritize clarity, security, and sovereignty (local-first where possible)
- Document decisions and trade-offs
- Respect privacy and ethical considerations in all AI and data work

### Communication
- Be respectful and constructive
- Discussions are welcome in Issues and (when enabled) Discussions
